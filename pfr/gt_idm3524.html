Function template gt

pfr::gt — Compares lhs and rhs for greater-than using their own comparison and conversion operators; if no operators available returns pfr::lt_fields (lhs, rhs).

Synopsis

// In header: <pfr/ops.hpp>


template<typename T, typename U> unspecified gt(const T & lhs, const U & rhs);

Description

Returns:

true if lhs is greater than rhs; false otherwise
